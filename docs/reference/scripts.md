# Scripts

## Orquestador

- `scripts/run_prjna717739_pipeline.sh`: ejecuta todo el flujo.

## Metadatos

- `scripts/parse_runinfo.py`: filtra RunInfo a Illumina paired-end.
- `scripts/get_leaf_order.py`: extrae orden de hojas del arbol.

## AMR

- `scripts/build_gene_classes.py`: genera tabla gen->clase.
- `scripts/build_amr_matrix.py`: crea matriz AMR y etiquetas MDR.

## Figuras

- `scripts/plot_tree_rectangular.py`: arbol rectangular con ETE3.
- `scripts/plot_amr_profile.py`: dot-plot AMR con brackets por clase.

## Almacenamiento

- `scripts/storage_report.sh`: reporte de consumo de disco por etapa.
