# AMR_plotting

Documentacion oficial del pipeline filogenomico de resistencia para *Klebsiella pneumoniae*.

Objetivo principal:
- Tomar lecturas crudas SRA paired-end.
- Estimar SNPs con Snippy para maxima resolucion.
- Inferir arbol filogenetico.
- Integrar resistoma (AMRFinderPlus) con filogenia de forma robusta.
